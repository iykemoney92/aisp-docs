# Account Setup

This guide will help you create and configure your AI Studio Plus account for the best experience.

## Creating Your Account

### Sign Up Methods

AI Studio Plus offers two ways to create an account:

#### 1. Email & Password

1. Go to [studio.aisp.app/gpt/signup](https://studio.aisp.app/gpt/signup)
2. Enter your email address
3. Create a password (minimum 6 characters)
4. Confirm your password
5. Click **Create Account**
6. Check your email for verification code
7. Enter the verification code to activate your account

:::tip Email Verification
Email verification is required before you can use the platform. Check your spam folder if you don't receive the email within 5 minutes. The verification code is valid for a limited time.
:::

#### 2. Google Sign-In

1. Click **Continue with Google**
2. Select your Google account
3. Grant permissions
4. Complete profile setup
5. You're ready to go!

### Account Verification

If you signed up with email and password:

1. Check your inbox for an email from AI Studio Plus
2. You'll receive a verification code
3. Enter the code on the verification page
4. Your email is now verified

:::info Resending Verification Code
Didn't receive the code? Click **Resend Verification Code** on the verification page. Wait a few moments before requesting another code.
:::

## Profile Setup

### Complete Your Profile

After signing up (especially with Google), you'll be guided to complete your profile:

#### Basic Information

1. **Display Name** (Required)
   - How you'll appear in the platform
   - Pre-filled from Google account if using Google Sign-In

2. **Email** (Auto-filled, read-only)
   - Your email address used for sign-up
   - Cannot be changed during profile setup

3. **Profile Picture** (Optional)
   - Upload an image (JPG, PNG, WEBP supported)
   - Maximum file size: 5MB
   - **Note**: HEIC/HEIF formats not supported
   - On mobile: Access camera or photo gallery directly

4. **Referral Code** (Optional)
   - Enter a referral code if you have one
   - Both you and the referrer will receive bonus credits
   - Code is validated before submission

5. **How Did You Hear About Us?** (Optional)
   - Select from dropdown menu
   - Helps us understand our user base

#### Skip Profile Setup

You can skip profile setup if you want to start using the platform immediately:
- Click **Skip for Now**
- You can complete your profile later from Account Settings
- Your account will still be marked as profile completed

:::tip Referral Rewards
Using a referral code during signup gives you bonus credits! Ask friends who are already using AI Studio Plus for their referral code.
:::

### Update Profile Later

To update your profile after initial setup:

1. Click your **profile icon** (top right in navigation)
2. Select **Update Profile**
3. Edit your information:
   - Display name
   - Profile picture
   - Preferences

## Security Settings

### Password Management

#### Change Your Password

1. Go to **Settings** > **Account**
2. Click **Change Password**
3. Enter current password
4. Enter new password (minimum 6 characters)
5. Confirm new password
6. Click **Update Password**

#### Password Requirements

Your password must be at least 6 characters long.

:::tip Strong Passwords
While 6 characters is the minimum, we recommend using longer passwords (12+ characters) for better security. Consider using a password manager to generate and store strong, unique passwords.
:::

### Forgot Password

If you can't remember your password:

1. Go to the sign-in page
2. Click **Forgot Password**
3. Enter your email address
4. Check your email for reset instructions
5. Click the link in the email
6. Create a new password
7. Sign in with your new password

## Troubleshooting

### Can't Verify Email

- Check spam/junk folder
- Make sure you're entering the correct verification code
- Click "Resend Verification Code"
- Wait a few moments between resend attempts
- Code may have expired - request a new one

### Forgot Password

1. Click **Forgot Password** on sign-in page
2. Enter your email address
3. Check email for reset link
4. Click link in email
5. Create a new password
6. Sign in with your new password

### Google Sign-In Not Working

- Make sure you're signed into the correct Google account
- Clear browser cache and cookies
- Try in incognito/private mode
- Disable browser extensions that might block popups
- Check that third-party cookies are enabled

### Profile Picture Upload Issues

- Ensure file is JPG, PNG, or WEBP format (HEIC not supported)
- File size must be under 5MB
- Try compressing the image
- On mobile, grant camera/photo access permissions

### Referral Code Not Working

- Check that the code is entered correctly (case-sensitive)
- Referral codes can only be used once per account
- Code must be from an active user
- Enter code during profile setup, not after
